Layanan Kami

Temukan layanan terbaik dari kami, dengan membuat solusi kreatif untuk mempererat ikatan antar sesama atau branding dengan media papan permainan.

Menyewakan Sarana Bonding Activities

Board Game sebagai sarana membangun interaksi sosial dan meningkatkan keakraban serta seru-seruan bareng tim, kerabat, dan rekan kerja.

Syarat & Ketentuan Peminjaman

Para pelanggan yang menggunakan layanan peminjaman wajib mematuhi syarat dan ketentuan berikut ini. Dengan mengajukan permohonan peminjaman, pelanggan dianggap telah membaca dan menyetujui semua ketentuan yang tercantum.

Kewajiban Pelanggan

  • Jika board game atau komponen rusak atau hilang setelah digunakan, pelanggan wajib menggantinya sesuai dengan nilai barang dalam rupiah.
  • Pengembalian yang tidak tepat waktu akan dikenakan denda sebesar Rp 35.000 hingga Rp 1.000.000. Tidak ada toleransi, bahkan jika terlambat hanya 1 menit. Nominal denda disesuaikan dengan lamanya keterlambatan pengembalian.
  • Pelanggan yang sengaja atau tidak sengaja menduplikat karya kami akan dikenakan sanksi sesuai dengan Undang-undang Hak Cipta.
  • Pelanggan yang melanggar syarat dan ketentuan di atas akan dilarang menggunakan layanan kami dalam jangka waktu tertentu.

Ketentuan Pembayaran:

  • Pembayaran wajib dilakukan sebelum layanan dapat diterima oleh pelanggan.
  • Pelanggan dapat menambah durasi penyewaan dengan membayar Rp 5.000 per kepala. Pembayaran dapat dilakukan melalui metode di atas atau secara tunai (diberikan kepada master board game).

Peminjaman dan Pengembalian(tanpa master)

  • Opsi Peminjaman
    • Isi Formulir Peminjaman: Klik disini
    • Hubungi via WhatsApp: Klik disini
  • Proses Peminjaman:
    • Pelanggan akan mendatangi lokasi usaha. Lokasi akan dibagikan melalui platform WhatsApp.
    • Sebelum menerima board game, pelanggan wajib menyerahkan kartu identitas asli (hanya KTM/KTP yang diperbolehkan).
    • Pastikan Board Game dikembalikan dalam kondisi utuh dan bebas cacat.
    • Pengembalian Board Game harus sesuai dengan jadwal yang telah disepakati.

Peminjaman dan Pengembalian(dengan master)

  • Opsi Peminjaman
    • Isi Formulir Peminjaman: Klik disini
    • Hubungi via WhatsApp: Klik disini
  • Proses Peminjaman:
    • Master akan mendatangi lokasi yang telah disepakati. Lokasi ini akan dibagikan oleh pelanggan melalui platform WhatsApp.
    • Ongkos Kirim:
      • Jarak 0-5 km: Gratis.
      • Jarak 5-10 km: Rp 10.000.
      • Jarak 10-15 km: Rp 20.000.
      • Jarak lebih dari 15 km: Kami hanya melayani penyewaan tanpa master.
    • Pelanggan diharapkan merespons dengan cepat pada hari yang telah disepakati untuk memudahkan pengiriman dan layanan dari Tutingtas Creation.
  • Tugas Master:
    • Master bertanggung jawab untuk menata board game sebelum permainan dimulai.
    • Master memberikan penjelasan tentang cara bermain board game.
    • Setelah permainan selesai, master mengemas kembali semua komponen board game.
    • Pelanggan berhak memanggil master jika ada kebingungan saat bermain board game.
  • Pengembalian Board Game:
    • Pengembalian board game harus sesuai dengan jadwal yang telah disepakati sebelumnya.

Pembelian Board Game dengan Berlangganan

  • Layanan berlangganan ini ditujukan untuk sebuah institusi, terkhusus institusi Pendidikan.
  • Layanan yang kami tawarkan ini hanya berlaku untuk board game yang kami terbitkan saja, untuk sejauh ini opsi anda hanya Veriy and Valor.
  • Apa yang institusi dapatkan? 1 set komponen main boardgame Verity and Valor dengan kustomisasi set kartu. *sesuai dengan kesepakatan pihak Tutingtas Creation dengan institusi pendidikan.

Opsi Berlangganan

  • Berlangganan 6 Bulan:
    • Konten dapat disesuaikan dengan permintaan institusi.
    • Institusi akan menerima satu set board game original buatan kami dengan satu set kartu kustom sesuai permintaaan institusi saat pengiriman set boardgame pertama *akan ditentukan tanggal pengiriman tiap set saat setelah terjadi kesepakatan berlangganan.
    • Institusi akan menerima paket kartu baru setiap 3 bulan sekali (Total 3 set) *+1 set kartu diawal kesepakatan berlangganan dimulai *+1 set kartu di 3 bulan pertama dan +1 set kartu di 3 bulan terakhir *update kartu akan diterima institusi maksimal minggu terakhir dibulan yang sudah ditentukan diawal kesepakatan.
    • Jasa pelatihan cara bermain board game akan disediakan disetiap jadwal pengiriman kartu. *pelatihan secara offline dilakukan saat pengiriman set boardgame pertama. *pengiriman set kartu berikutnya hingga set kartu terakhir akan difasilitasi pelatihan secara online, berupa video instruksi.
  • Berlangganan 12 Bulan:
    • Konten dapat disesuaikan dengan permintaan institusi.
    • Institusi akan menerima satu set board game original buatan kami dengan satu set kartu kustom sesuai permintaaan institusi saat pengiriman set boardgame pertama *akan ditentukan tanggal pengiriman tiap set saat setelah terjadi kesepakatan berlangganan.
    • Institusi akan menerima paket kartu baru setiap 3 bulan sekali (Total 5 set) *+1 set kartu diawal kesepakatan berlangganan dimulai *+1 set kartu di 3 bulan pertama, +1 set kartu di 6 bulan berjalan, 1 set kartu di 8 bulan berjalan, 1 set kartu di 12 bulan berjalan. *update kartu akan diterima institusi maksimal minggu terakhir dibulan yang sudah ditentukan diawal kesepakatan.
    • Jasa pelatihan cara bermain board game akan disediakan disetiap jadwal pengiriman kartu. *pelatihan secara offline dilakukan saat pengiriman set boardgame pertama *pengiriman set kartu berikutnya hingga set kartu terakhir akan difasilitasi pelatihan secara online, berupa video instruksi.

Biaya Berlangganan

  • Biaya berlangganan akan ditentukan berdasarkan durasi berlangganan dan kebutuhan kustomisasi konten.
  • Pembayaran dilakukan total 2 kali:
    • uang muka : 50 % dari estimasi biaya total dari riset dan produksi *ditetapkan di awal kesepakatan pembelian secara berlangganan board game.
    • tagihan tersisa : dibayarkan bersamaan dengan pengiriman set board game dan set kartu pertama.

Pengiriman dan Pengembalian

  • Pengiriman board game dan paket kartu baru akan dilakukan setiap 3 bulan sekali ke alamat yang telah disepakati.
  • Jika terdapat kerusakan pada paket kartu saat diterima, institusi berhak melaporkannya dalam waktu 7 hari kerja untuk melakukan ajuan penggantian.

Pelatihan dan Dukungan

  • Jasa pelatihan cara bermain board game akan dilakukan oleh tim kami di tempat (saat pengiriman set boardgame pertama) atau secara daring (pengiriman set kartu berikutnya hingga set kartu terakhir).
  • Dukungan teknis dan konsultasi akan tersedia selama masa berlangganan.

Kustomisasi Konten

  • Permintaan kustomisasi harus disampaikan saat sebelum terjadi kesepakatan berlangganan dimulai.
  • Akan dilakukan riset untuk keperluan kustomisasi konten disetiap set kartu, diharapkan institusi dapat memberikan data yang lengkap terkait tema permintaan kustomisasi secara tepat waktu demi kelancaran riset dan produksi.
  • Akan dilakukan riset keselutruhan terkait isi konten untuk setiap set kartu diawal setelah terjadinya kesepakatan pembelian berlangganan. *untuk ajuan penggantian isi konten set kartu setelahnya, institusi diwajibkan melakukan ajuan perubahan minimal 3 bulan sebelum kartu set selanjutnya dikirim. *Biaya tambahan mungkin dikenakan untuk melakuakan penggantian isi konten.

Pembatalan Berlangganan

  • Institusi dapat membatalkan pembelian berlangganan dengan pemberitahuan tertulis 1 bulan sebelum dikirimkan set boardgame yang pertama.
  • Pembelian berlangganan tidak dapat dibatalkan setelah dikirimkan set boardgame yang pertama, biaya yang telah dibayarkan tidak dapat dikembalikan.

Privasi dan Keamanan Data

  • Kami berkomitmen untuk menjaga privasi dan keamanan data institusi.
  • Data yang diberikan oleh institusi hanya akan digunakan untuk keperluan layanan berlangganan dan tidak akan dibagikan kepada pihak ketiga tanpa izin.

Perubahan Syarat & Ketentuan

  • Kami berhak untuk mengubah syarat dan ketentuan ini dengan pemberitahuan sebelumnya kepada institusi.
  • Perubahan akan berlaku efektif setelah pemberitahuan dan institusi dapat memilih untuk melanjutkan atau membatalkan berlangganan berdasarkan syarat dan ketentuan yang baru.

Dengan berlangganan layanan kami, institusi menyetujui semua syarat dan ketentuan yang telah ditetapkan. Jika terdapat pertanyaan lebih lanjut, silakan hubungi nomor yang tertera di bawah.
